# blogging-project

to run the project setup the .env file in the backend 

DATABASE_URL="....."
install the npm packages 
then run npm run dev in the backend
cd backend
npm install
npm run dev

run npm i in frontend as well 
run npm run dev
cd frontend
npm install
npm run dev
